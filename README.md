**注意：您不可以直接用于生产环境，因为背景图片是由[picsum.photos](https://picsum.photos/400/300)提供的。您需要自行寻找其他CDN服务商。**  
如下所示：
```html
  <script>
    // 配置参数

      bgWidth: 320,       // 背景图宽度

      pieceSize: 50,      // 拼图块尺寸
```
